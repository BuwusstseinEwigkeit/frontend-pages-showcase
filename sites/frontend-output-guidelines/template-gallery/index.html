红线验收 Checklist

生成任何 HTML 文件后，逐条检查以下红线。本画廊所有文件均已通过。

红线规则

R1 禁止外链资源 — 不得引用 CDN / Google Fonts / 外部 JS/CSS，字体用系统栈 ✓
R2 圆角 ≤ 8px — border-radius 最大 8px，禁止 12px 及以上 ✓
R3 不用泛紫渐变 — 禁止 linear-gradient 中出现紫–蓝–粉过渡色 ✓
R4 不用装饰 blob — 禁止绝对定位 + 大尺寸 + blur 做背景装饰 ✓
R5 单文件交付 — 所有 CSS/JS 内联，可直接双击打开 ✓
R6 中文默认 — lang="zh-CN"，所有界面文本为中文 ✓

通用检查项

A1 响应式：375px 宽度无横向滚动 ✓
A2 可访问性：有 lang 属性，表单控件有关联 label ✓
A3 无 alert()、confirm()、prompt() 弹窗 ✓
A4 无未使用的全局变量或空函数 ✓

常见反例

以下是生成 HTML 时最常见的违规写法，对照红线逐条避免。

R1 外链字体 — 最常见违规

<link href="https://fonts.googleapis.com/css2?family=Inter" rel="stylesheet">
<link href="https://cdn.jsdelivr.net/npm/..." rel="stylesheet">

修复：改用系统字体栈 -apple-system, "Microsoft YaHei", sans-serif，不引用任何外部资源。

R2 圆角过大 — 视觉失控

border-radius: 16px;
border-radius: 9999px;

修复：最大 8px。按钮 4px，卡片 4–6px，头像等圆形元素除外（可用 9999px 但需注明是圆形头像场景）。

R3 紫色渐变 — 泛滥审美

background: linear-gradient(135deg, #667eea, #764ba2);
background: linear-gradient(to right, #a855f7, #ec4899);

修复：用纯色或蓝/青/灰等中性色系，保持信息优先。

R4 装饰 blob — 增加视觉噪音

.blob {
  position: absolute;
  width: 300px; height: 300px;
  border-radius: 50%;
  filter: blur(80px);
  opacity: 0.3;
}

修复：删除所有装饰性背景元素。页面信息密度优先，不需要"艺术感"填充。

R1 外部 JS — 单文件原则

<script src="https://cdn.jsdelivr.net/npm/vue@3"></script>
<script src="https://unpkg.com/react@18"></script>

修复：所有 JS 内联在 <script> 标签中，纯原生实现，不依赖任何框架。

A3 alert 弹窗 — 低级交互

function copy() {
  alert('已复制到剪贴板');
}

修复：用 navigator.clipboard.writeText + 自定义 toast 提示，不用浏览器原生弹窗。

控制台模板 vs 演示模板

两套模板面向不同场景，以下是关键维度对比。

维度 控制台模板 演示模板
场景 工程监控、运维仪表盘、数据看板 汇报演示、周报、可编辑幻灯片
背景色 浅灰 #f1f3f5 — 高信息密度可读 深色 #0f172a — 投影沉浸感
布局 CSS Grid 3列，卡片跨列(span-2/3) 单页 section 居中，全屏覆盖
顶栏 44px sticky topbar，状态指示灯+实时时钟 无顶栏，全屏沉浸
底部导航 无（信息密度优先） 48px fixed 底栏，上一页/下一页/页码/复制
可编辑 不可编辑（展示数据） contenteditable="true"，hover 显示虚线边框
数字字体 monospace，语义色（绿/黄/红） sans-serif，中性色
典型组件 指标格子、进度条、键值表、日志面板、标签 标题、列表、代码块、页码水印
交互 <details/> 折叠、按钮组、实时日志滚动 键盘←→导航、圆点指示器、复制页面HTML
响应式 桌面3列 → 平板2列 → 手机1列 标题字号随屏幕缩小，小屏隐藏编辑提示

选型建议

控制台 需要展示多组实时数据、状态、日志，信息密度是第一优先级
演示 需要逐页展示内容，内容可编辑，面向投屏/会议场景
混合 可从两套模板各取组件组合，统一使用 CSS 变量保证风格一致
